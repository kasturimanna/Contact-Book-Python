# Contact Book Management System (Python)

This is a simple menu-driven Contact Book application developed using Python.
It allows users to add, view, and search contact details using file handling.

## Features
- Add new contact
- View all contacts
- Search contact by name
- Exit the application

## Technologies Used
- Python
- File Handling
- Functions
- Modules

## Project Structure
contact.py   - Contains all contact-related functions
main.py      - Main menu-driven program
contact.txt  - Stores contact information

## How to Run
1. Make sure Python is installed
2. Keep all files in the same folder
3. Run the program using:
   python main.py
